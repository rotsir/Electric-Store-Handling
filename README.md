# Electric Store Handling System ⚡

Electric Store Handling System is a complete web-based solution designed to manage an electrical store efficiently. This project helps in managing products, services, customer inquiries, and store information through a structured and user-friendly interface.

## Features
- Product management system
- Services listing (installation, repair, maintenance)
- Admin dashboard
- Inventory handling
- Contact form for customers
- Responsive design
- Clean UI structure
- Organized file architecture

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap (optional if you used it)

## Project Structure
- Home page
- About page
- Services pages
- Product management section
- Contact page
- Admin panel

## Purpose
This project was created for learning purposes and portfolio demonstration. It shows frontend development skills, UI design, and basic management system architecture.

## Future Improvements
